# Lending Club Loan Analysis

## Overview
This project analyzes Lending Club loan data to predict default risk using machine learning.

Dataset:
https://www.kaggle.com/datasets/wordsforthewise/lending-club

## Problem
Can we predict whether a borrower will default?

## Project Structure
- data/ → raw data (not included)
- notebooks/ → EDA
- src/ → pipeline code
- outputs/ → results

## Setup
```bash
pip install -r requirements.txt
pip install -r requirements.txt```

## Models
- Logistic Regression
- Random Forest
- XGBoost (optional)

## Metrics
- AUC
- Accuracy
- Precision/Recall
